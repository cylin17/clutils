# clutils

## Install
1. Add it in project's build.gradle
   ```
    allprojects {
        repositories {
            ...
            maven { url 'https://jitpack.io' }
        }
    }
   ```
2. Add the dependency
   ```
    dependencies {
        implementation 'com.github.cylin17:clutils:Tag'
    }
   ```
3. Enjoy!
